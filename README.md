#Synced

Synced is a thin wrapper script around `rsync` that allows you to keep the contents of two folders in sync.

##Dependencies

* Ruby 1.9.* or later
* OSX or Linux system with `rsync`

##Getting Started

* Download or clone the project and `cd` into the project directory
* Type `ruby synced.rb`
